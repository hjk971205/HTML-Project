Zombie Trail

Project Overview

A mock-up version of The Oregon Trail game but zombie edition. The user can select 4 team members to travel in a van to reach their SafeHaven destination. While in the game, the user has to make some smart choices by keeping their team members alive, otherwise when a team member gets sick, they can die!

    Bakery

    Project Overview

    The user will be able to interact with the terminal to order bread and pastry. The special deal for bread is that with every orders of 2 bread, the third will be free. The special deal with the pastry is that with every 3 pastries, the fourth will be free. Then the application should be able to send out the calculated amount of money for the order inputted.

    Technologies Used

  • C#
  • .NET
